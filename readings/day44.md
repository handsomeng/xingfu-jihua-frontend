# 换个角度看坏事

## 一、钩子

想象你刚刚搞砸了一次重要的客户提案。客户当场冷脸，老板事后找你谈话。

现在你心里冒出来的那句话，更像下面哪一个。

A：「我做事就是不行，什么场合都搞不定，这次又是我的问题。」

B：「这次提案没准备好，是这个项目这个环节的失误，时机加上准备不够一起造成的。」

事实是同一件事。变的只是你脑子里那句话怎么写。

Seligman 在《习得性乐观》里讲过一个结论：同样这件事，按 A 那么说的人，24 小时之后还在低气压。按 B 那么说的人，2 小时就能回到工位接着干。

差别藏在 3 个开关里。今天讲怎么手动拨这 3 个开关。

## 二、讲清楚

Seligman 做了 30 年研究，把人对坏事的解释方式拆成 3 个独立的维度。每一个都是一组对照，悲观一边，乐观一边。

**第一个开关：时间长度。**

悲观的说法是「我永远做不好这件事」。乐观的说法是「这次没准备好」。

关键词在时间。一次性的事，被悲观的说法拉成了永久状态。同一场搞砸的提案，前者会让人接下来 1 周不敢约下一个会，后者第二天就能继续约。

**第二个开关：覆盖范围。**

悲观的说法是「我什么都不行」。乐观的说法是「这门课不行」。

关键词在范围。单一场景的失误，被悲观的说法扩散到了整个自我，或者整个生活领域。前者会让一次销售失败影响到家庭、健身、社交全部领域。后者只让它停在那一单上。

**第三个开关：责任归属。**

悲观的说法是「都是我的错」。乐观的说法是「这次时机不对，加上我准备可以更充分」。

关键词在归因。悲观的说法把所有责任全揽到自己身上。乐观的说法承认外部因素存在（时机、配合方、市场行情、客观条件），同时保留自己能改进的那一部分。

这一条最容易被误会。乐观的说法不等于推卸责任。它要求你诚实分清楚哪部分是自己的，哪部分是外部的。

Seligman 1980 年代跟大都会人寿做过一个真实业务实验。他给数千名销售员候选人测这 3 个维度的分数。结果：3 个开关都偏乐观的人，次年留任率比偏悲观的高出约一半，第二年业绩比传统能力测试达标的那一组还要高出 27%。

机制很简单。销售是个高拒绝率岗位。被拒 10 次之后还能继续打电话的人，跟被拒 3 次就放弃的人，差别就在这 3 个开关。乐观的人把每次拒绝解释成「这一通时机不对、这家公司不需要、我的话术可以再调」。悲观的人把每次拒绝解释成「我永远做不了销售、我什么客户都搞不定、全是我的问题」。3 个一叠加，第 4 通电话就打不出去了。

这 3 个开关反过来用在好事上，结论刚好掉头。

乐观的人对好事的解释是「我一直都有这个本事」「我做事都还行」「我自己努力得来的」。悲观的人对好事的解释是「运气好这一次」「就这件事走运」「都是别人帮的」。

合起来看：悲观的人把坏事放大成永久、普遍、自己的，把好事缩小成暂时、特定、别人的。每天都在双向折损。乐观的人反过来。每天都在双向积累。10 年下来差距会很大。

好消息是，这 3 个开关可以手动调。心理学里有个词叫认知重评，简单说就是把那句自动冒出来的悲观话写下来，逐个开关拨一遍。

拨的时候有 3 个分寸要把握。

第一，贴近事实。把「我永远做不好销售」改成「我特别优秀，是这些公司没眼光」就过了。这种叫盲目正能量，跟认知重评是两回事。正确的改写应该是「最近这 8 家公司不合适，简历版本可以再调」。承认有失误，承认外部因素，承认自己能改进，3 个开关都收紧到事实里。

第二，逐个找关键词。

写下那句自动出现的悲观话，逐个开关找：

有没有「永远 / 一直 / 从来 / 这辈子」？改成「这次 / 最近 / 这段时间」。

有没有「什么都 / 所有 / 全部 / 我这个人」？改成「这件事 / 这个场景 / 这种类型」。

有没有「都是我 / 全怪我」？老实问外部因素占几成，把那部分还回去。

3 个开关都拨完后，再把新句子重新念一遍。

第三，注意身体反应。

好的改写念完之后，肩颈和呼吸会有一点松动。如果改写完身体没反应，或者更紧绷，说明改写要么没贴近事实（你自己不信），要么没真正拨到那个关键的开关。身体是这件事的免费验证器。

## 三、它和你有什么关系

这 3 个开关对你的实际价值在 3 个地方。

第一，决定你从坏事里多久爬出来。同样被批评一次，3 个开关全偏悲观的人，接下来 24 小时心里都在循环「我永远 / 我什么都 / 都是我」。3 个开关都偏乐观的人，2 小时之后已经在改方案。这是回弹速度的差距。

第二，决定你敢不敢再试一次。被拒一次就放弃的人，多数把这次拒绝解释成永久 + 普遍 + 个人。Seligman 那个销售员研究就证明了这条机制。同样的拒绝量，乐观的人能撑到出单，悲观的人第 3 通电话就出局。

第三，决定你会不会把好事也错过。3 个开关反过来用在好事上结论翻转。悲观的人把坏事放大、把好事缩小。乐观的人反过来。一天一天累加，10 年后两种人活成的样子差距很大。

这 3 个开关是能练的，不是天生定死的。

## 四、今天可以怎么做

今天花 5 分钟做一次。

把最近 7 天里一件具体的坏事写下来，再把你当时心里冒出来的第一句话也写下来。

逐个开关找关键词，按上面给的方法把 3 个开关全部拨到乐观这一边。

把改写后的新句子写下来。要贴近事实，不要假大空。

把旧句子和新句子对比着念 3 遍，留意身体反应。

做完去卡片流 p24 记一下。

身体是质量的验证器，不放过这一步。

## 五、一句话收尾

> 解释方式决定回弹速度。同样的坏事，悲观的人循环 24 小时，乐观的人 2 小时回到工位。差别就在 3 个开关怎么拨。
